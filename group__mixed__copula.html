Detailed Description

The mixed-discrete normal copula. The method is from
Smith, Michael & Khaled, Mohamad. (2011). Estimation of Copula Models With Discrete Margins via Bayesian Data Augmentation. Journal of the American Statistical Association. 107. 10.2139/ssrn.1937983.

Bernoulli marginal

Bernoulli marginal for mixed continuous-discrete Gaussian copula.

The lb and ub: if y == 0, upper bound at inv_Phi( y, mu ) if y == 1, lower bound at inv_Phi( y - 1, mu )
